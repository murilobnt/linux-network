## Linux Network

This project is written in Python, and it uses the Matplotlib, Pandas, NetworkX
and the nxviz libraries.

## How to run

Initial requirements:
* Python 3
* Pip

```sh
python3 -m pip install -r requirements.txt
python3 linux-network.py
```
